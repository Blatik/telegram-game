# 💰 Financial Calculator Pro

## ✅ Готовый отдельный финансовый калькулятор!

### 🧮 8 профессиональных инструментов:

**Базовые (из Life Tools Hub):**
- ⏰ **Real Hourly Income** - реальный доход за час
- 💎 **Time Value** - стоимость времени
- 🏦 **Credit Decision** - анализ кредитов  
- 🏠 **Buy vs Rent** - покупка vs аренда

**Новые инструменты:**
- 📈 **Investment Growth** - рост инвестиций с ежемесячными взносами
- 🏖 **Retirement Plan** - планирование пенсии (правило 4%)
- 💳 **Debt Payoff** - стратегия погашения долгов
- 📊 **Tax Calculator** - расчет налогов по странам
